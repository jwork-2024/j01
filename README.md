1. 《葫芦兄弟》视频来自B站（例如example中采用 https://www.bilibili.com/video/BV1Fk4y1V7Lf ）；
2. 选取桥段应尽量精彩；

4. 请定义github actions运行scene的main方法获得输出；
5. 注意gitignore问题；
6. 编写README文件；
7. 请参考example下示例完成改作业。
